Интеллектуальный анализатор и заполнитель форм

Скрипт выполнен с использованием Browser Automation Studio.
Входные данные - список страниц с формами.
Суть работы - автозаполнение полей форм, включение чекбоксов и радиокнопок, кодирование и отправка ответа на сервер.

Входные данные

Входными данными является список сайтов.

Настройки программы

Перед запуском задается множество параметров, таких как составной креатив, количество потоков, число выполнений.
Также была реализована гибкая архитектура для вызова функции в несколько потоков внутри каждого.

Схема скрипта

Процесс работы скрипта представляется набором блоков и функций исполняемых скриптом заданное число раз.
В данной программе эмуляция браузера не используется, вся работа идет через http клиент с помощью библиотеки CURL.
Некоторые части реализованы на чистом javascript, что дает гибкость, скорость и наглядность.

Процесс работы

В процессе работы программа запускает потоки заданным количеством повторов - в данном случае, используются прокси http/socks5, ведется пулл проксей.
Архитектура работы программы меняется в зависимости от входных настроек. Реализован механизм отбора и тестирования прокси, автозавершения при потере интернета и т.д.

Результат работы

Результат работы - лог действий в csv формате.

Компиляция программы

После завершения разработки программу можно запускать шаблоном в формате xml из среды BAS, либо скомпилировать в исполняемую.
При этом при первом запуске скачается набор библиотек для ее исполнения размером около 500 Мб.
Программа работает в MS Windows 7,8,10 x86/x64. Также возможна работа в эмуляторе Wine в Linux.
