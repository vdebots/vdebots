Парсер объявлений и контактов с Авито

Скрипт выполнен с использованием Browser Automation Studio.
Входные данные - категория для парсинга.
Суть работы - скрипт переходит по страницам категории и берет ссылки на объявления.
После этого в многопоточном режиме скрипт проходит по ссылкам с предыдущего этапа и парсит информацию.
В формате csv сохраняются заголовок, описание, цена и номер телефона.

Настройки программы

Перед запуском задается ссылка на категорию и вариант работы программы.

Схема скрипта

Процесс работы скрипта представляется набором блоков и функций.
Так как программа работает с мобильной версией сайта, используется отпечаток сотового телефона.
Эмулируется мобильный браузер, в который загружаются категории, а после объявленя.
По факту загрузки извлекается код страницы и с помощью адресации селекторов xPath достаются требуемые данные.

Процесс работы

За счет использования мобильной версии загрузка происходит быстро, а номер не требует OCR обработки.

Результат работы

Результат работы - данные по объявлениям в csv формате.

Компиляция программы

После завершения разработки программу можно запускать шаблоном в формате xml из среды BAS, либо скомпилировать в исполняемую.
При этом при первом запуске скачается набор библиотек для ее исполнения размером около 500 Мб.
Программа работает в MS Windows 7,8,10 x86/x64. Также возможна работа в эмуляторе Wine в Linux.
