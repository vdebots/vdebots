Парсер позиций сайтов с сервиса "spymetrics" и их конкурентов.

Скрипт выполнен с использованием Browser Automation Studio.
Входные данные - список доменов.
Выходные - таблица с SEO позициями по каждому домену в csv формате.

Входные данные

Входными данными является список доменов для которых необходимо узнать SEO позиции.

Настройки программы

Так как программа выполнена многопоточной, перед запуском задается количество процессов, которое будет запущено.
Также указывается файл со списком доменов и создается хранилище для результатов работы программы.
Дополнительно предусмотрена возможность замены отпечатков устройства с целью уникализации ПК для снятия ограничений по железу сайтом.

Схема скрипта

Процесс работы скрипта описывается набором блоков, которые последовательно выполняются.

Процесс работы

В процессе работы программа запускает потоки заданным количеством повторов - в данном случае, пока не завончатся все домены.
После этого составляется адрес и загружается сайт для парсинга, при этом сайт воспринимает скрипт как полноценный браузер.
Также можно добавить случайную активность в виде движения мыши для более полной эмуляции человека.

Для извлечения данных программа берет код страницы и находит в нем необходимые css селекторы в html коде.
Также возможна обработка извлеченных данных.

Результат работы

Результат работы можно записать в любом текстовом формате, будь то csv, txt, sql формат.

Компиляция программы

После завершения разработки программу можно запускать шаблоном в формате xml из среды BAS, либо скомпилировать в исполняемую.
При этом при первом запуске скачается набор библиотек для ее исполнения размером около 500 Мб.
Программа работает в MS Windows 7,8,10 x86/x64. Также возможна работа в эмуляторе Wine в Linux.
